Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafo
  • Título
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores

    • Lista com demarcadores são elementos que servem para marcar ou destacar itens. No contexto de HTML, se refere às listas com marcadores, ou seja, listas com bolinhas, quadradinhos etc. (como as <ul>).

    Quando você faz uma lista usando <ul>, aparecem bolinhas (•) na frente de cada item, certo? Essas bolinhas são chamadas de demarcadores, em inglês, markers.
